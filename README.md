 🎧 简易听力/视频分段标记助手 (Zero-Dependency Media Marker)
一个简洁、高效的本地化工具，帮助你为长时音频或视频文件添加自定义书签和时间标记点。
最大特点：本程序仅使用 Python 标准库，无需安装 Streamlit 或其他任何第三方依赖！
✨ 主要功能特点
  * **零依赖运行**：程序仅依赖 Python 环境自带的库（`http.server`, `webbrowser`, `json`），运行前**无需**执行 `pip install` 命令。
  * **自定义书签**：支持为文件添加任意数量的自定义时间标记点（例如：Part 1, 重点难句, 5:00）。
  * **一键跳转**：点击标记列表中的按钮，媒体播放器将立即跳转到该时间点，方便重复练习。
  * **数据持久化**：标记数据通过后端服务器保存在本地 **JSON 文件**中，下次打开同一文件时，标记依然存在。
  * **Web UI 界面**：通过 Python 内置的 HTTP 服务器和浏览器提供界面，用户体验良好。
  * **多媒体支持**：兼容浏览器支持的 MP3、WAV、MP4 等多种音频和视频格式。
## 🚀 技术栈
本项目是一个轻量级的 **C/S (客户端/服务器)** 结构应用，但所有组件都运行在本地：
| 领域 | 技术 | 描述 |
| :--- | :--- | :--- |
| **后端/数据** | **Python 标准库** (`http.server`) | 核心服务器和数据处理，实现数据持久化。 |
| **前端/UI** | HTML / CSS / JavaScript | 界面和播放器逻辑，无需任何外部依赖。 |
| **数据存储** | JSON | 本地化存储标记数据，确保数据持久性。 |
## ⚙️ 安装与运行
本项目最大的优势就是**无需安装**，只需具备 Python 3 环境即可。
### 1\. 运行程序
进入到你保存脚本（例如 `audio_marker_lite.py`）的目录，直接运行：
python audio_marker_lite.py
### 2\. 访问应用

程序启动后，你会看到类似 **`✅ 服务已启动！`** 的提示，它会自动在你的默认浏览器中打开应用（通常是 `http://localhost:9999`）。

**注意：** 在程序运行期间，请勿关闭运行 Python 脚本的命令行窗口，因为它就是后台服务器。

## 📖 使用指南
1.  **上传文件**：点击界面上方的 **“选择文件”** 按钮，选择你的音频或视频文件。
2.  **定位时间**：在播放器中播放文件，找到你需要标记的关键时间点。
3.  **添加标记**：
      * 播放器下方会显示 **“当前时间”**。
      * 在输入框中输入 **标记名称** (如：`Part 2`)。
      * 点击 **“添加标记”** 按钮。
4.  **快速跳转**：标记会出现在右侧的列表中。点击标记的文字区域，媒体播放器即可跳转到该时间点开始播放。

## ❌ 如何停止应用？

直接关闭运行 `audio_marker_lite.py` 脚本的**命令行窗口**即可停止服务器。
